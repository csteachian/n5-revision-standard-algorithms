What is Input Validation?

Input validation is a technique used to ensure that data entered by the user falls within an acceptable range before it is processed by the program.

Key Points:

  • ‚Ä¢ Uses a conditional loop (WHILE loop) to repeatedly ask for input
  • ‚Ä¢ The loop continues while the input is invalid
  • ‚Ä¢ An error message should be displayed when invalid input is detected
  • ‚Ä¢ The loop only exits when valid input is provided

üí° Exam Tip

Remember: The condition checks for invalid data (e.g., "while age < 0 OR age > 120"), not valid data!

üìã Pseudocode 

RECEIVE age FROM KEYBOARD

WHILE age < 0 OR age > 120 DO
    SEND "Invalid age. Please enter a value between 0 and 120" TO DISPLAY
    RECEIVE age FROM KEYBOARD
END WHILE

SEND "Valid age entered: " & age TO DISPLAY

üêç Python Code 

age = int(input("Enter your age: "))

while age < 0 or age > 120:
    print("Invalid age. Please enter a value between 0 and 120")
    age = int(input("Enter your age: "))

print("Valid age entered:", age)

üîÑ Traversing a 1D Array

What is Traversing an Array?

Traversing means visiting every element in an array, one at a time, usually to search for a value, display all items, or perform an operation on each element.

Key Points:

  • ‚Ä¢ Uses a fixed loop (FOR loop) to visit each element
  • ‚Ä¢ The loop runs from 0 to length-1 (in Python)
  • ‚Ä¢ Each element is accessed using array[index]
  • ‚Ä¢ Commonly used for linear search to find a specific value

üí° Exam Tip

When searching, use a Boolean variable (found) to track if the item was located. Set it to False initially and True when found!

üìã Pseudocode (Linear Search) 

SET names TO ["Alice", "Bob", "Charlie", "Diana", "Eve"]
RECEIVE searchName FROM KEYBOARD
SET found TO FALSE

FOR index FROM 0 TO LENGTH(names) - 1 DO
    IF names[index] = searchName THEN
        SET found TO TRUE
        SEND "Found at position " & index TO DISPLAY
    END IF
END FOR

IF found = FALSE THEN
    SEND "Name not found" TO DISPLAY
END IF

üêç Python Code 

names = ["Alice", "Bob", "Charlie", "Diana", "Eve"]
search_name = input("Enter name to search: ")
found = False

for index in range(len(names)):
    if names[index] == search_name:
        found = True
        print("Found at position", index)

if not found:
    print("Name not found")

‚ûï Running Total

What is a Running Total?

A running total is a technique where values are accumulated as they are entered. Each new value is added to a total variable that keeps track of the sum.

Key Points:

  • ‚Ä¢ The total variable must be initialised to 0 before the loop
  • ‚Ä¢ Uses total = total + value to accumulate
  • ‚Ä¢ Can use either a FOR loop (fixed count) or WHILE loop
  • ‚Ä¢ The final total is displayed after the loop ends

üí° Exam Tip

Always remember to initialise total = 0 BEFORE the loop! This is a common mistake in exams.

üìã Pseudocode 

SET total TO 0

FOR counter FROM 1 TO 5 DO
    RECEIVE number FROM KEYBOARD
    SET total TO total + number
END FOR

SEND "The total is: " & total TO DISPLAY

üêç Python Code 

total = 0

for counter in range(5):
    number = int(input("Enter a number: "))
    total = total + number

print("The total is:", total)

üìä Running Total with 1D Array

What is Running Total with an Array?

This combines traversing an array with a running total. Instead of getting values from user input, we sum all the values that are already stored in an array.

Key Points:

  • ‚Ä¢ Combines traversal and running total techniques
  • ‚Ä¢ Total is initialised to 0 before the loop
  • ‚Ä¢ Uses total = total + array[index] to accumulate
  • ‚Ä¢ Loop runs from 0 to length-1 to visit every element

üí° Exam Tip

This algorithm is often used to calculate averages - just divide the total by the length of the array after the loop!

üìã Pseudocode 

SET scores TO [85, 92, 78, 95, 88]
SET total TO 0

FOR index FROM 0 TO LENGTH(scores) - 1 DO
    SET total TO total + scores[index]
END FOR

SET average TO total / LENGTH(scores)
SEND "Total: " & total TO DISPLAY
SEND "Average: " & average TO DISPLAY

üêç Python Code 

scores = [85, 92, 78, 95, 88]
total = 0

for index in range(len(scores)):
    total = total + scores[index]

average = total / len(scores)
print("Total:", total)
print("Average:", average)
